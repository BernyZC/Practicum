* Portafolio
Bienvenido a mi repositorio de prácticas. Aquí documento mi progreso en el aprendizaje de tecnologías web, desde la experiencia de usuario hasta la lógica de programación dinámica.

* Detalle de Prácticas
Práctica 1: Usabilidad de Páginas Web
Aprendizaje: Análisis de la experiencia de usuario (UX). Se trabajó en la jerarquía visual, facilidad de navegación y accesibilidad para asegurar que el sitio sea intuitivo.

Práctica 2: HTML
Aprendizaje: Fundamentos de la estructura web. Creación de documentos semánticos utilizando etiquetas esenciales para definir el esqueleto de un sitio.

Práctica 3: Gestión de Datos con Tablas
Aprendizaje: Organización de información compleja de manera tabular. Aprendí a estructurar datos para que sean legibles y visualmente ordenados.

Práctica 4: Formularios Dinámicos
Aprendizaje: Creación de interfaces interactivas para la captura de datos. Implementación de validaciones y elementos que reaccionan a la entrada del usuario.

Práctica 5: Sistemas de Registro
Aprendizaje: Integración de componentes para la gestión de usuarios. Enfoque en el flujo de datos desde la entrada del usuario hasta la simulación de almacenamiento.

Práctica 6: Implementación Multimedia
Aprendizaje: Integración de elementos visuales y auditivos para enriquecer la experiencia del usuario sin sacrificar el rendimiento de carga.

*Medio Término: Desarrollo de una Cuenta Regresiva Dinámica
Descripción: Proyecto integrador donde se aplicó lógica de programación para crear un contador en tiempo real.
